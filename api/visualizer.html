Parameters:

  • time_scale ‚Äì Visualization timescale (speed-up factor).

  • space_scale ‚Äì Visualization space scale (should be small, to reduce numbers to a manageable level).

  • viewport_size ‚Äì Size of the viewport (width, height).

Initialize the visualizer.

Parameters:

  • constellation ‚Äì The constellation to visualize.

  • show_isls ‚Äì Whether to show ISLs.

  • time_scale ‚Äì Visualization timescale (speed-up factor).

  • space_scale ‚Äì Visualization space scale (should be small, to reduce numbers to a manageable level).

  • host_radius ‚Äì Radius of the host body (e.g. Earth). If None, no host body will be rendered.

  • host_rotation_period ‚Äì Rotation period of the host body (in seconds).

  • viewport_size ‚Äì Size of the viewport (width, height).

  • sat_color ‚Äì Color of the satellites.

  • isl_color ‚Äì Color of the ISLs.

  • host_materials ‚Äì Tuple of paths to the host body‚Äôs diffuse, normal, and metallic-roughness maps.

  • host_color ‚Äì Color of the host body.

  • bg_color ‚Äì Background color.
